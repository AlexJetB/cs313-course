Overview

For this week, we will begin our study of JavaScript and Node.js.

Infrastructure Setup

Your first preparation task this week is to download and install Node.js on your local computer. This will enable you to run JavaScript files directly, and as we progress, you will host a Web server with it.

Next week, we will also configure a Node.js server at Heroku, but for this week, you only need to install it locally.

Here is the link to the Node.js website: nodejs.org. Please follow their instructions to download and install on your operating system.

Reading

If you are new to Node.js or advanced JavaScript programming, one of the biggest things that takes some getting used to is the idea of asynchronous programming and callback functions.

To help you get started, after setting up your infrastructure. Please read the following:

  1. The Art of Node : Callbacks - The whole thing is good (and not too long), but make sure to read at least the section on Callbacks.

  2. Callback Hell

Depending on your familiarity with JavaScript from previous classes, you may also want to spend some time reviewing the fundamentals of JavaScript in preparation for this week.
