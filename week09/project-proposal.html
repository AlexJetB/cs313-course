09 Prove: Project 2 Design

Overview

Please see the overview of Project 2 from last week's material for more information about this project.

Your assignment this week is to plan out the components of your project.

Submission

Please submit to I-Learn a document containing the following sections:

  1. Project Description. In 1-2 paragraphs, describe the overall application.

  2. Data Layer. Describe the data layer of your application. If it consists of a DB, please list the tables you will use along with their columns and relationships.

  3. Web service endpoints. List the web service endpoints your Node.js application will expose. For each one, specify the information it will need from the client and the data it will return. Depending on your application, this may be very similar to your data layer, with your services essentially providing CRUD (Create, Retrieve, Update, Delete) operations for your DB entities.

  4. User Interface. Describe the UI of your application. Describe the way the AJAX calls will manipulate the HTML page. Mockups may be very helpful here.

  5. User Accounts / Sessions. If applicable, briefly describe the role that user accounts and/or sessions will play in your application.
