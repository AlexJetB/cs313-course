Overview

This week you will learn about two things:

  1. Express.js, a Node.js Framework that will simplify our efforts.

Please note that Express is used much more broadly than just with EJS pages. This week, we will see how to use EJS templates, but in future weeks, we will also see how to use Express to send back responses in other ways (e.g., JSON).

EJS

You can think of EJS templates in many ways like the PHP pages that we have used before. We create HTML and then can embed JavaScript commands in the HTML to render our variables, or loop through a list of results, etc.

A major potential difference with using a templating engine like EJS and the way that many of us used PHP, is that we really want to have our EJS templates be restricted to "Views" from the Model-View-Controller design pattern. Briefly stated, the purpose of the template is to render the data, not to perform computation, tasks, or run logic. The idea is that we do all of the logic first in our Node.js functions, and then pass the results on to the template to render them.
