04 Prove : Assignment - DB Setup

Overview

Create the database for your PHP Project.

Guidelines

For this assignment, you need to have the schema in place for your database (e.g., create the tables and the relationships between them). You are not required to have the tables populated with data just yet (although, depending on your application, it might be a good idea to get some of this in place sooner than later).

Please consider the following guidelines regarding this assignment:

  • Create a database on your development machine and on your server.

  • Create the tables you will need for your project.

  • Choose appropriate data types for each column.

  • Ensure that each table has an appropriate primary key.

  • Include foreign key constraints wherever possible.

  • Eliminate data redundancy wherever possible.

Demonstrate your database design by showing the tables, describing each one, and then showing the create statements necessary to create each. To do so, run the following commands and then copy and paste the results into a text document to submit to I-Learn.

SHOW TABLES

Then for each table:

DESCRIBE your-table-name;

SHOW CREATE TABLE your-table-name;

where “your-table-name” is the name of the table.

Submission

Finally, download this assignment's submission form, fill it out, and submit it to I-Learn.
