Topics for Review

Here is a list of topics that you should be comfortable discussing at this point. For each topic you should understand the mechanics (i.e. how to use it), but more importantly, understand the higher level concepts of each such as the proper use, strengths, limitations, etc.

  • HTTP concepts (headers, request, response, etc.)

  • Session tracking concepts (HTTP state, cookies, sessions, etc.)

  • Database design (normalization basics, constraints, foreign keys, etc.)

  • Database queries/modification

  • PHP database interaction

  • Java database interaction

  • JSPs, servlets, and the interaction between them

  • Basic JSTL and scriptlets

  • Object-relational mapping

  • Web services (Consuming REST, OAuth basics, etc.)

  • Design patterns such as: Model View Controller, n-Tier Architecture, Data Access Objects

  • Security concerns (SQL injection, XSS, handling passwords)
