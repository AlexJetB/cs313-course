11 Teach : Team Activity

Special Instructions for Online Students

You need to coordinate a one hour synchronous meeting with your group for this activity.

Objective

Understand how to use client-side Javascript to retrieve data from a web service and render it on the page.

To focus on the process of retrieving this data on the client, for this activity we will not be retrieving data from our own Node.js servers, but rather an existing service. But please recognize that the process is the same, and we could retrieve data from a service such as the one we built in last week's team activity. That is what you will be doing for your projects.

Overview

For today's team activity, we will create a simple web application to search for movies, using the API provided by Open Movie Database API. You should have an HTML page with a search box, and, when the user clicks the search button, use AJAX to process the search, call the API, and display a list of movies on the page, without causing the page to refresh. As a stretch challenge, when displaying the list of movies, put a "view details" option next to each one that will use AJAX to again query the API, this time for information about that specific movie, and then render the details on the page, again without causing a page refresh.

You are welcome to use JavaScript libraries such as jQuery to help with the AJAX call and the DOM manipulation, if you would like.

Instructions

  1. Create an HTML page with a search box and a button. When the button is clicked, use client side JavaScript to get the value from the search box, and log it to the console.

  2. Use AJAX to access the OMDB API for the provided search value. When the result comes back, log it to the console to verify that the API call is working.

  3. Use client side JavaScript to update the HTML page to show the list of movies, without causing the page to refresh.

Stretch Challenges

After finishing the core requirements, ensure that everyone is at that point and understands the material. When everyone has completed the core requirements, you can move on to these stretch challenges.

  1. Add a "View Details" link or button next to each movie result in the list. If this button is clicked, log the movie ID to the console.

  2. Add an AJAX call for your View Details functionality to call into the OMDB API for that particular movie. When the result is retrieved, log it to the console.

  3. Use client side JavaScript to update the page and display more information about the movie including an image and more detailed information as provided by the API.
