11 Teach : Team Activity - Bonus!

Objective

This week you have a bonus team activity! The objective of this activity is to bring together to points of the previous two activities (creating a web service in Node.js, and consuming it via client-side AJAX to update an HTML page.

Overview

For this activity we will work with a database of scriptures as we did earlier in the semester.

The assignment is to create an HTML that displays the scriptures in the database and can filter them by the book name. If you have time, add another page that allows the addition of a new scripture to the database using a post method.

Instructions

  1. Create a scripture table in the database and insert some scriptures. You are welcome to use one from before, or to use the scripture from the instructor's solution to Team Activity 06.

    Then, create a new Node.js application using Express. Create an endpoint to retrieve all the scriptures in your database, and another way to get all the scriptures for a particular book. (For example, you could have a route /scriptures that returns all the scriptures if no additional parameters are passed, but also accepts an optional parameter for book, that returns the scriptures matching that book name.)

    Verify that your Web service can be called an that it returns the correct JSON.

  2. Create an HTML web page in your static pages section of your Node.js application that initially gets all the scriptures.

    Then, include a search box and a button that allows the user to enter a book and then uses AJAX to get the new filtered list of books and update the list on the page.

  3. Create a second page for scripture entry. Provide inputs for the user to add a book, chapter, verse, and content, and then a submit button. When the user submits this data, use post method to send it to your Node.js server.

    Please note that this will require you to get the data a different way, and will require a little bit of digging to learn how to do this using Express.

Submission

This is an ungraded activity to help you on your projects.
