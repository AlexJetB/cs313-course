Overview

In our previous lesson, we learned about Web services from the server perspective. In this lesson, we now shift our attention to how these services are consumed. While these services can be used in many different ways, one of the most common is to be called from client-side AJAX functions that then manipulate the DOM on the HTML page.

The following reading will introduce these concepts to you, but it is anticipated, that you will also need to dig deeper into certain aspects of this as you complete your assignment for the week, which is to write AJAX functions to call into the Web services you created last week.

This week, we will consider this task for APIs that do not require authentication, which as you might imagine, is a much simpler case. Next week, we will look more deeply at how to do this when authentication is needed.

Preparation Material

There are two components to consuming these APIs to update the web page: first, making the AJAX calls, and second, updating the web page by manipulating the HTML DOM.

AJAX

You should have been introduced to using AJAX in your previous client-side programming course. As a review, please read over the following resources concerning AJAX:

  1. AJAX - Mozilla Developer Network

  2. The jQuery .ajax function

  3. From StackOverflow: What is the difference between XMLHttpRequest, jQuery.ajax, jQuery.post, jQuery.get

  4. Then, please review this heavily commented A simple Node.js application that demonstrates the use of AJAX to consume Node.js services.

DOM Manipulation

Just as you can perform AJAX calls using different libraries, you can also manipulate the HTML DOM in different ways. Here are some resources that demonstrate how to do this using both Vanilla JavaScript (in other words, no outside libraries) and also using the popular jQuery library. You are welcome to use either approach for your projects.

First, using Vanilla JavaScript:

  • document.querySelector()

  • document.getElementById()

  • document.getElementsByClassName()

  • document.getElementsByTagName()

  • Node.appendChild()

  • Node.removeChild()

  • Node.innerText

  • Element.innerHTML

And here using jQuery:

  • ID Selector (“#id”)

  • Class Selector (“.class”)

  • Element Selector (“element”)

  • .append()

  • .remove()

  • .text()

  • .html()
