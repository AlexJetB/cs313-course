CS 313 | Web Engineering II

10 Prepare : Reading

Overview

This week, you will learn about how to query a PostgreSQL database in Node, return the data back to a client, and how to configure the client to access this data via AJAX and update the corresponding page.

Preparation Material

The preparation material for this week has you look a little bit at what web services are, and then at some elements of getting data from PostgreSQL in Node.js.

  1. An overview of what a REST API is and some best practices to keep in mind.

  2. Node pg module - The module we will use to connect to PostgreSQL and run our queries.
