Special Instructions for Online Students

You need to coordinate a one hour synchronous meeting with your group for this activity.

Overview

Use JPA to create a simple Java application that lists authors and the books they have written. For simplicity, we will assume that each book was only written by one author (but authors can write multiple books). Also, for simplicity, you may write this as a standalone Java application (not a web app).

Instructions

Using JPA, list all the authors in the DB with the books they have written. Then, using JPA, in your Java code, create a new author, and a new book written by that author, add them to the database, and show the list again.

Instructor's Tip

Make sure to reference the Students/Majors example in this week's preparation material as it is very similar to your assignment.

Core requirements

  1. Create a new PostgreSQL database for your authors and books. Add a table for each of these entities and create a user with permissions to this DB.

    Insert a few authors and books into your DB.

  2. Using JPA, retrieve each Author from the DB and display them to the screen.

  3. For each author, retrieve their books and display them to the screen as well.

Stretch Challenges

After finishing the core requirements, ensure that everyone is at that point and understands the material. When everyone has completed the core requirements, you can move on to these stretch challenges.

  1. Using JPA, add a new Author to your database.

  2. Again, using JPA, add two books to an author and persist them to the database.

  3. Create a new class for Genre that contain a genre name (e.g., Science Fiction, Fantasy, Historical Fiction, etc). Allow books to have a genre. Use JPA to automatically create the corresponding table in the database.
