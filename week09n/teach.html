CS 313 | Web Engineering II

09 Teach : Team Activity

Special Instructions for Online Students

You need to coordinate a one hour synchronous meeting with your group for this activity.

Objective

Understand the fundamentals of Node.js and Express.

Overview

Create a Web application that performs basic Math operations. This application begins with a static form that asks the user to select a basic math operation (add, subtract, multiply, or divide) and two operands. When the form is submitted the user is shown a results page that displays the data that was submitted and the result.

Instructions

  1. Create a new Node.js application. You may want to do this by cloning the Heroku Node.js Getting Started Git repository, and beginning from there. After cloning the repository, you'll want to run "npm install" in the application directory, to have it download any dependencies (i.e., express).

    Create a static HTML page for your submission form. You may want to refer to the section on serving static files from the Express documentation for this.

    Have your form action go to a route "/math" with the operation and the operands passed as parameters (it is probably easiest to begin with handling a Get method).

    Verify that you can load your form in a browser (at localhost).

  2. Use Express to handle this route and connect it to a function (or functions) that will obtain the parameters, and compute the result. Then, report the correct result on the console and verify that everything is working.

    At this point, focus on implementing the add function. You can complete the others as part of the stretch challenges.

  3. Create a results page that is an EJS. Pass the result value from your Node.js JavaScript function to the EJS and have it render the result.

    This may require you to search the Web a little about rendering data in an EJS. Also, you can reference how the Getting Started repo handles the request for "/" and uses the index.ejs file to render a response.

    Verify that your app is working correctly.

Stretch Challenges

After finishing the core requirements, ensure that everyone is at that point and understands the material. When everyone has completed the core requirements, you can move on to these stretch challenges.

  1. Add the functionality for the other operations (subtract, multiply, divide).

  2. Add another route to your server, "math_service" that also computes the result of the math operation, but this time returns the result as JSON, rather than a formatted HTML document.

  3. Add another button on your form (not a submit button) that uses AJAX to call your new math_service, obtain the result, and then update the DOM of the current page to display it.
