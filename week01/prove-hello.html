CS 313 | Web Engineering II

01 Prove : Hello World Assignment : Deploy a Hello World Page

Create a Hello World Page

In the web subdirectory, create your own hello.html page that displays the text "Hello World." You are free to use a text editor of your choice. In this example we are using the Sublime Text editor (subl).


$ cd cs313-php

$ subl web/hello.html

Commit the Hello World Page

Use the git status command to see your changes and then use the git add and commit commands to include your changes to the local repository


$ git status
  On branch master
  Your branch is up-to-date with 'origin/master'.
  Untracked files:
  (use "git add <file>..." to include in what will be committed)

  web/hello.html

  nothing added to commit but untracked files present (use "git add" to track)

$ git add web/hello.html

$ git status
  On branch master
  Your branch is up-to-date with 'origin/master'.
  Changes to be committed:
  (use "git reset HEAD <file>..." to unstage)

    new file:   web/hello.html

 $ git commit -m "added hello world page"
    [master ae7f2f9] added hello world page
    1 file changed, 5 insertions(+)
    create mode 100644 web/hello.html

Open the Hello World Page

Use the generated URL or the heroku open command to view your application in a browser and append your new "/hello.html" file to the URL (e.g. https://afternoon-mesa-40442.herokuapp.com/hello.html) to see your new hello world page deployed in the Cloud.
