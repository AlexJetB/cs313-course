Heroku CLI

The Heroku Command Line Interface (CLI) is a tool for creating and managing Heroku applications.

Install the Heroku CLIS by following the installation instructions on the Heroku CLI page. (Just install the CLI for now, we will be creating the application in future steps.)

Verify your installation by running the version command and login with the credentials you used to create your Heroku account.


$ heroku --version
heroku-toolbelt/3.43.13 (x86_64-darwin10.8.0) ruby/1.9.3
heroku-cli/5.4.8-eee5ec9 (darwin-amd64) go1.7.1
    
$ heroku login
Enter your Heroku credentials.
Email: lyonr@byui.edu
Password (typing will be hidden):
Authentication successful.

PHP 5.6

Install the latest 5.6.x version of PHP on your local machine by following the PHP Installation Instructions page.

Verify your installation by running the version command.

Composer

Install Composer on your local machine by following the Getting Started page.

Verify your installation by running the version command.


$ composer -V
Composer version 1.1.3 2016-06-26 15:42:08

Git

Install Git on your local machine by running the installation package from the Git Downloads page.

Verify your installation by running the version command.


$ git --version
git version 2.8.4 (Apple Git-73)

Congratulations

You now have all the tools in place to create your first Heroku application!
