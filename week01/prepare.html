Summary

Welcome to Week One of CS 313!

This week you will be learning about the course, getting to know each other, creating cloud platform accounts, setting up our development environments, and following the time honored "Hello World!" tradition by creating an HTML file and deploying it to an application in the cloud.

Git and GitHub

For this class, we will dive right in to Git and assume that you either have a working knowledge of it, or can come up to speed quickly. If you are a little uneasy about Git, here are some resources that may be helpful for you:

  • Git Video Tutorial - This is a great video tutorial that introduces Git.

  • Version Control with GitHub - This is the introduction to Git from CS 246. It also contains links of the place to go to sign up for a free student account.
